# SMR BoT

> To run this nodejs server CREATE 'config.env' file in config folder.

> ##### *Variables needed,*
>
> - PORT
> - MONGO_URI
> - TWILIO_ACCOUNT_SID
> - TWILIO_AUTH_TOKEN
> - TWILIO_NUMBER
> - GOOGLE_CLIENT_ID
> - GOOGLE_CLIENT_SECRET
> - SESSION_SECRET
> - GMAIL
> - GMAIL_PASSWORD
>

>  ***Redis*** is required.
